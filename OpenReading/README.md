### OpenReading

**OpenReading** 是 OpenResearch 项目下的一个论文分析与阅读模式，基于 GitHub 上的 Discussion、Issue、PR、Wiki 等功能，及其这些实体之间的转化。目的是迭代一个开放、高效、能够不断沉淀的分享模式。
